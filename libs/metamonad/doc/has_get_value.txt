has_get_value
=============
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class T>
struct has_get_value
{
  // unspecified
};
--------------------------------------


Description
-----------

Check if a +T::get_value+ function is available. Returns a wrapped boolean
value.

Header
------

--------------------------------------
#include <mpllibs/metamonad/has_get_value.hpp>
--------------------------------------

Example
-------

--------------------------------------
struct foo {};

assert(!has_get_value<foo>::type);
--------------------------------------
